## Installation 💻

**❗ In order to start using Ew Format you must have the [Dart SDK][dart_install_link] installed on your machine.**

Add `ew_format` to your `pubspec.yaml`:
```yaml
dependencies:
  ew_format:
```

Install it:
```sh
dart pub get
```
---

## Running Tests 🧪
To run all unit tests:
```sh
dart test
```
